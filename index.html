Параметры обработки
Консистентные маски Одно значение → одна маска во всём файле
Читаемые метки (USER_0001 вместо XXXX) Маски содержат категорию и порядковый номер
Сохранять даты и статусы без изменений Поля типа created_at, status, type не затрагиваются
Очищать метаданные файла Автор, организация в DOCX/PDF/XLSX
Журнал замен Показывать список что→что изменено
Пользовательские паттерны
Работает только при шаблоне Пользовательский или дополняет любой шаблон.
Каждое правило — пара Regex → Метка. Совпадение будет заменено на МЕТКА_0001, МЕТКА_0002 и т.д.

Примеры паттернов:
\bCONTRACT-\d+\b — номера договоров CONTRACT-12345
\bMY_ID_\d{6}\b — внутренние ID MY_ID_000123
[A-Z]{2}\d{6} — коды формата AB123456
(?i)(?:server|host)=\S+ — hostname в конфигах
