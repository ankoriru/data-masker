Задача Создай веб-приложение для маскирования чувствительных данных в корпоративных файлах. Технологии React + TypeScript + Vite Tailwind CSS + shadcn/ui Библиотеки: xlsx, jszip, pdf-lib Поддерживаемые форматы Логи: LOG, CSV, JSON, XML, TXT Системные: SAP Audit, Siebel Audit, Documentum Audit, Syslog, Apache/Nginx Документы: XLSX, DOCX, PDF Категории для маскирования (выбор через UI) Table Категория Что скрывается Пример Персональные данные ФИО, email, телефоны ivanov@mail.com → user_0001@company.com Коммерческая информация Суммы, клиенты, контракты 1 000 000 ₽ → AMOUNT_0001 Технические данные IP, UUID, ID сессий 192.168.1.1 → 10.0.0.1 Внутренняя информация ID пользователей, документы USER_001 → USER_0001 Шаблоны систем SAP CRM — маскирует GUID, BNAME, KUNNR, TCODE Siebel CRM — маскирует Row ID, бизнес-объекты Documentum — маскирует Object ID, workflow Универсальные логи / Веб-сервер / База данных / Пользовательский UI структура Загрузка файла — drag & drop Вкладка "Шаблоны" — выбор системы Вкладка "Категории" — выбор что маскировать (4 карточки с чекбоксами) Вкладка "Детально" — расширенные настройки Результат — редактор с оригиналом/результатом + кнопка скачать Ключевые правила Для текста: Замена на читаемые значения (USER_0001, не XXX) Одинаковые значения → одинаковые маски в файле Даты, статусы, типы НЕ трогать Для DOCX: Распаковать ZIP → обработать XML → запаковать обратно Маскировать текст в элементах Обработать document, headers, footers, comments Для XLSX: Читать через библиотеку xlsx Маскировать значения ячеек Сохранять форматирование Для PDF: Удалять метаданные (автор, заголовок) Сохранять оригинальное содержимое Тексты интерфейса Заголовок: "Маскирование документов и логов корпоративных систем" Подзаголовок: "Защита чувствительных данных" Кнопки: "Загрузите файл", "Обфусцировать", "Скачать документ" Важно Интерфейс на русском Все ресурсы в бандле (без внешних CDN) При ошибке обработки — возвращать оригинал Показывать toast-уведомления
